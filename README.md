# RSS агрегатор

Сервис для агрегации RSS-потоков, удобного чтения статей в одном месте.

## Demo

[Ссылка на приложение](https://fullstack-javascript-project-11-dun.vercel.app/)

## Установка

```bash
npm ci
```

## Запуск в режиме разработки

```bash
npm run dev
```

## Сборка для production

```bash
npm run build
```
